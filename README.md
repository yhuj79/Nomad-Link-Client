## :file_folder: 프로젝트 개요

* 명칭 : Nomad Link
* 개발 인원 : 2명
* 개발 기간 : 2022.07.26 ~ 2022.09.15
* 주요 기능 : 개인 회원 로그인, 개발자/기업 리스트 조회, 이력서 작성, 기업 폼 작성, 채용 공고에서 지원
* 개발 환경 
  * Visual Studio Code, IntelliJ
  * React 18.2.0 / Spring Boot 2.7.2 / Java 11 / H2 Database
* 형상 관리 툴 : Git
